Завдання №1
Створіть клас «Дріб» (або використовуйте вже раніше створений вами). Використовуючи перевантаження
операторів реалізуйте для нього арифметичні операції
для роботи з дробом (операції +, -, *, /).
Завдання №2
Створіть клас Complex (комплексне число) або використовуйте вже створений вами клас. Створіть перевантажені оператори для реалізації арифметичних операцій
для роботи з комплексними числами (операції +, -, *, /).
