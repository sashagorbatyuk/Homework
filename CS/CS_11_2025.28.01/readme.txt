Завдання: Робота із серіалізацією та атрибутами в C#
Опис завдання
Реалізуйте програму для серіалізації та десеріалізації інформації про книги. При цьому:

Використайте атрибути для налаштування серіалізації.
Реалізуйте серіалізацію в JSON і XML форматах.
Використайте атрибути для виключення певних полів із серіалізації.
Вимоги до реалізації
Клас Book:

Містить поля:
Title (назва книги, string).
Author (автор, string).
Price (ціна, decimal).
InternalId (внутрішній ID книги, int, виключити із серіалізації).
Використайте атрибути, щоб:
Змінити назву властивостей у JSON/XML (наприклад, Title → BookTitle).
Виключити InternalId із серіалізації.
Методи:

Метод для серіалізації колекції книг у JSON.
Метод для серіалізації колекції книг у XML.
Метод для десеріалізації з JSON.
Метод для десеріалізації з XML.
Головний клас Program:

Створіть кілька об’єктів Book і додайте їх до списку.
Виконайте серіалізацію цього списку в JSON і XML.
Запишіть результати у файли.
Зчитайте дані з файлів і виконайте десеріалізацію.
Виведіть десеріалізовані дані в консоль.
