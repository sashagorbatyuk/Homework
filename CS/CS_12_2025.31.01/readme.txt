Завдання 1: Фільтрація студентів
У вас є список студентів, представлений у вигляді об'єктів:

class Student
{
    public string Name { get; set; }
    public int Age { get; set; }
    public double Grade { get; set; }
}
Створіть список студентів.
Використовуючи LINQ, знайдіть студентів старше 20 років із середнім балом вище 85.
Відсортуйте результати за спаданням середнього балу.
Виведіть імена студентів та їхні оцінки.

Завдання 2: Підрахунок слів у тексті
Напишіть програму, яка:

Зчитує текст із файлу.
Використовуючи LINQ, підраховує кількість унікальних слів.
Виводить кожне слово разом із кількістю його повторень у тексті.
Завдання 3: Робота з товарами
Уявіть, що у вас є список товарів:

class Product
{
    public string Name { get; set; }
    public string Category { get; set; }
    public decimal Price { get; set; }
}
Створіть список товарів.
Знайдіть найдешевший та найдорожчий товари в кожній категорії.
Обчисліть середню ціну товарів у кожній категорії.
Виведіть результати на екран у зручному форматі.

Завдання 4: Статистика про числа
Створіть масив випадкових чисел (наприклад, 50 чисел у діапазоні від 1 до 100).
Використовуючи LINQ:
Знайдіть парні числа.
Обчисліть суму непарних чисел.
Підрахуйте кількість чисел, більших за середнє значення масиву.

Завдання 5: Аналіз продажів
У вас є список замовлень:

class Order
{
    public string CustomerName { get; set; }
    public DateTime OrderDate { get; set; }
    public decimal TotalAmount { get; set; }
}
Створіть список замовлень.
Використовуючи LINQ, знайдіть:
Замовлення, зроблені у поточному місяці.
Сумарну суму замовлень за кожен місяць.
Клієнта, який зробив найбільше замовлень.
Виведіть результати.

Завдання 6: Пошук у файлі
Зчитуйте текст із файлу (можна використовувати .txt).
Знайдіть усі рядки, які містять задане слово (наприклад, "LINQ").
Виведіть номери рядків та сам текст рядків, у яких знайдено це слово.

Завдання 7: Групування міст
У вас є список міст:


class City
{
    public string Name { get; set; }
    public string Country { get; set; }
    public int Population { get; set; }
}
Згрупуйте міста за країнами.
Знайдіть країну з найбільшою загальною кількістю населення.
Виведіть назви міст із населенням понад 1 мільйон, відсортовані за спаданням.
